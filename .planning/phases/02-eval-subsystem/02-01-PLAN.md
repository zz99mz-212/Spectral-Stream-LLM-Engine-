## Phase Goal

**As a** researcher evaluating SpectralStream compression, **I want to** run WikiText-2 perplexity on original versus compressed weights and get a JSON artifact with a recovery gate, **so that** I can independently verify that quality is preserved.

*(Derived from ROADMAP.md Phase 2 goal, which is outcome-shaped but not in user-story format.)*

<objective>
Deliver the end-to-end eval grader (Slice 1): a runnable CLI that measures WikiText-2 perplexity on a base model and a compressed `.ssf`, computes the recovery ratio, enforces the gate, and emits a JSON artifact with all D-09 fields.

Purpose: Closes the project's single biggest trust gap by proving quality preservation on real weights with honest, measured perplexity.
Output: `eval/run_eval.py` and supporting modules, plus a passing `tests/test_eval_grader.py` suite.
</objective>

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-01-01 | Tampering | `eval/model_path.py` `resolve_model_path` | high | mitigate | Reuse `cli.py` `_PATH_TRAVERSAL_PATTERN` regex to reject `..` segments; resolve and verify file existence before returning |
| T-02-01-02 | Denial of Service | `spectralstream/inference/pipeline.py` `measure_perplexity` | high | mitigate | Replace full-vocab exp with vocab-blocked log-softmax (`VOCAB_LOG_SOFTMAX_BLOCK_SIZE = 4096`); load base and compressed models sequentially with `close()` between |
| T-02-01-03 | Tampering | `eval/artifact.py` `build_eval_artifact` | high | mitigate | `recovery_ratio` and `gate_passed` are computed only from the two measured PPLs; artifact schema records both raw values so the derivation is auditable |
| T-02-01-04 | Tampering | `eval/grader.py` `grade` | medium | mitigate | Record `layers_loaded` from `len(pipe.tensor_names)` and abort/warn if it is below the expected layer count; sequential loading guarantees only one model's weights are measured at a time |
| T-02-01-05 | Information Disclosure | `scripts/fetch_eval_corpus.py` | low | accept | Downloads public-domain WikiText-2 from a known URL using stdlib only; no credentials or secrets |
</threat_model>
