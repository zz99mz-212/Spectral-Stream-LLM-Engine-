## Phase Goal

**As a** developer running tests or using SpectralStream without a model tokenizer, **I want to** call `BaseTokenizer().encode("hello")` and have it round-trip through `decode`, **so that** the engine is usable out of the box without raising `NotImplementedError`.

<objective>
Fix the abstract-base tokenizer so it ships with a byte-identity fallback implementation and add a test proving encode/decode round-trips. This is the default tokenizer slice (EVAL-02).

Purpose: Removes the `NotImplementedError` blocker and provides a deterministic, always-available tokenizer for tests and offline eval.
Output: Patched `spectralstream/utils/tokenizer_engine.py` and passing `tests/test_tokenizer_fallback.py`.
</objective>
